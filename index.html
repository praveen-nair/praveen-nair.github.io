 Hybrid Script Server (v 1.0.1)
Programming or creating web sites should be easy. Often, too much time is spent on selecting a server to host the Web pages, and then a programming language to process the business logic. Hybrid script server takes focus of both these points. Hosting static page is as simple as copying and pasting the html resourse inside the application folder (web_root). To embed application logic, you need not stick to a single language. Choose between C/C++ or Python / Lua script to create the application logic to host your pages.

Download the binaries from here.

Unzip the contents to your local directory and execute HybridServer.exe
